What comes after install FreeBSD stable on the RPi

Here is some useful tips on creating a new user. Having a user account is helpful because you can have access when connected to a network. Now you can work on FreeBSD with sudo privileges with a regular user. Also, there will be an account you can actually work with over SSH.

  • Connect the RPi to a monitor and login as root.
  • Dispatch the command "useradd", this will provide simple interactive prompt.
  • While entering your arguments make sure to add the user to the wheel group.
